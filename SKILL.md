# 一键智能专利生成系统

## 简介

用户级别的专利自动生成Skill。只需提供一个专利题目，系统将自动完成：

1. **技术领域检测** - 自动识别专利所属技术领域
2. **全球专利检索** - 检索相关现有技术
3. **智能内容生成** - 利用专利专家知识生成完整专利内容
4. **自我审核改进** - 无限迭代直到通过审核
5. **高清图表生成** - 1200 DPI专业专利图表
6. **完整文档输出** - 生成可直接提交的Word文档

## 功能特性

- **一键生成**: 仅需提供专利题目，无需其他输入
- **自动检索**: 内置专利检索模块，检索全球专利数据库
- **智能审核**: 多维度审核（完整性、权利要求、新颖性、格式）
- **无限迭代**: 自动根据审核意见修改，直到通过为止
- **高清图表**: 生成1200 DPI专业专利图表
- **完整文档**: 生成符合中国专利局标准的Word文档

## 使用方法

### 方式一：在Skill目录运行

```bash
cd H:\智能体技能合集\good_skills-main\patent-one-click
python patent_one_click.py "专利题目"
```

### 方式二：在任意目录运行（推荐）

```bash
# 复制整个patent-one-click目录到你的项目
cp -r H:\智能体技能合集\good_skills-main\patent-one-click\ 你的项目目录\

# 运行
cd 你的项目目录\patent-one-click
python patent_one_click.py "专利题目"
```

### 方式三：指定输出目录

```bash
python patent_one_click.py "专利题目" -o "C:\输出目录"
```

## 专利文档结构

生成的Word文档包含以下部分：

1. 发明名称
2. 技术领域
3. 背景技术
4. 发明内容（技术问题、技术方案、有益效果）
5. 具体实施方式
6. 权利要求书（11条权利要求）
7. 附图说明

## 审核标准

系统自动按照以下标准审核专利：

| 维度 | 权重 | 说明 |
|------|------|------|
| 完整性 | 25% | 必要章节是否齐全 |
| 权利要求 | 25% | 独立/从属权利要求结构 |
| 新颖性 | 25% | 与现有技术的区别 |
| 格式规范 | 25% | 专利局格式要求 |
